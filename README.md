# 🎬 Proyek Analisis Data Film Terbesar
**Analis Data:** Yang Mulia Mas Rayyan 👑

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk mengolah data mentah perfilman dunia guna menemukan 10 film dengan pendapatan (*revenue*) tertinggi secara akurat.

## 🛠️ Proses yang Berhasil Dilakukan:
1. **Data Ingestion**: Berhasil memasukkan data Excel mentah ke dalam sistem Python menggunakan library `Pandas`.
2. **Data Cleaning**: Mendeteksi dan membersihkan data rusak (`ParserError`) serta membuang data duplikat (kasus film *Hoppers* yang terinput ganda).
3. **Data Visualization**: Menyajikan data peringkat dalam bentuk Grafik Batang Horizontal yang estetik dan mudah dipahami menggunakan `Matplotlib` dan `Seaborn`.
